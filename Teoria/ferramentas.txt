Express: framework NodeJS para aplicações WEB

EJS: Uma linguagem de modelagem para criação de paginas HTML utilizando JavaScript

Nodemon: Reinicia automaticamente o servidor NodeJS quando houver qualquer alteração nos Scripts

NPM: Gerenciador de pacores JavaScript

MÓDULOS: Permitem organizar melhor o código de forma a isolar determinada lógica e utilizar essa lógica de forma recorrente dentro da aplicação,
        se acemelha as classes de linguagens orientadas a objetos.

COMMONJS: È um formato de implementação dos módulos.

CONSIGN: Facilita o desenvolvimento de aplicativos com separação lógica de arquivos e carregamento automático de scripts. Pode ser usado para carregar automaticamente modelos, rotas, esquemas, configurações, controladores, mapas de objetos, etc.
